GooderPrint:



GooderPrint is a lightweight Python module for \*\*colorful terminal output\*\* and clean printing utilities.

It provides simple functions to print messages with different styles, such as warnings, errors, success messages, and debug info.





Installation:



You can install directly from GitHub:

```bash

pip install git+https://github.com/uzaygurgen55-del/GooderPrint.git

```





Usage:



```python

import GooderPrint as gp



gp.print\_warning("This is a warning!")

gp.print\_error("This is an error!")

gp.print\_success("Operation successful!")

gp.print\_debug("Debug info")

```





Features:



1\. Simple and clean syntax

2\. Color-coded messages for terminal

3\. Minimal dependencies (`colorama`)





Requirements:



\* `Python 3.6+`

\* `colorama >= 0.4.6`
